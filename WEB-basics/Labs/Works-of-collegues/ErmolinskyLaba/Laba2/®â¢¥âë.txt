1. Селектор - элемент CSS, который указывает на элемент на странице.

2. Селектор типа отбирает элемент по тегу
	класса - по классу
	id - по d

3. Контекстные селекторы - селекторы через пробел
 Дочерние - через >
 Смежные - через +

4. Селекторы псевдоклассов активируются при каком-то действии пользователя. 
Например,  hover активируется при наведении курсора на элемент.

5. Приоритет определяется степенью вложенности контекстных селекторов. 
Либо если поставить флажок !important,
либо если прописать стили непосредственно в тег элемента.

7. Классы - если хочешь задать стили нескольким элементам
   Идентификаторы - для одного элемента.
