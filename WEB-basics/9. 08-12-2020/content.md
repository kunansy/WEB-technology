## Условия и циклы в JS

### Условия

#### if, else if, else
```javascript
if (<condition>) {
    <do sth>;
} else if (<condition>){
    <do sth>;
} else {
    <do sth>;
}
```

#### switch
```javascript
switch (<condition>) {
    case <value>:
        break;
    default:
        break;
}
```


### Циклы
#### while
```javascript
while (<condition>) {
    <do sth>;
}
```


#### do while
```javascript
do {
    <do sth>;
} while (<condition>);
```


#### for
```javascript
for (let i = 0; i < 10; ++i){
    <do sth>;
}
```
